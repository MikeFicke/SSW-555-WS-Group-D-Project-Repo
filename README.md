# SSW-555-WS-Group-D-Project-Repo
Group D's project repo for SSW-555-WS.

## Overview
A command-line Python program that parses GEDCOM genealogy files, pretty-prints summaries of individuals and families, and detects errors in the data and logs them.

## Setup
```bash
# Create and activate virtual environment
python -m venv venv
.\venv\Scripts\activate        # Windows
source venv/bin/activate       # macOS/Linux

# Install dependencies
pip install prettytable
```

## Usage
```bash
python pretty_print.py <path_to_gedcom_file>
```
